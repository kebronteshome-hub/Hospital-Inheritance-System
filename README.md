# Hospital-Inheritance-System
# Description
This project is a simple hospital management system i created using Java and Object-Oriented Programming (OOP) concepts. It shows how inheritance works using different types of patient cases.

## What the program does
- Stores basic person and patient details
- Uses different patient types (InPatient, OutPatient)
- Displays information using methods
- Demonstrates inheritance concepts

## OOP Concepts Used
- Inheritance (Person → Patient → InPatient / OutPatient)
- Multilevel Inheritance
- Hierarchical Inheritance
- Method Overriding
- super keyword
- protected variables
- final method
- static method hiding... as we were instructed to.

## The Structure
- Person.java (base class)
- Patient.java (extends Person)
- InPatient.java (extends Patient)
- OutPatient.java (extends Patient)
- Hospital.java (final class)
- Main.java (runs the program)

## How to run it
1. Open the project in IntelliJ IDEA
2. Make sure all files are inside the "hospital" package
3. Run Main.java
4. Output will appear in the console
